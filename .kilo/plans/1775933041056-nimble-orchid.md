# Streambert Feature Implementation Plan

## Features to Implement

### 1. Auto-Play Next Episode
**Goal**: When episode ends, automatically play next (no new UI)
**Implementation**:
- Listen for video "ended" event in webview/custom player
- Auto-select next episode and call playEpisode
- Settings toggle to enable/disable auto-play
- Skip if last episode in season

### 2. Skip Intro for Live-Action
**Goal**: Skip intro/outro for non-anime content
**Implementation**:
- Create a generic skip API integration (different from AniSkip)
- Could use chapters/timestamps from TMDB video metadata
- Add skip buttons to player overlay (same as AniSkip but for any content)

### 3. Better Recommendations
**Goal**: More personalized "Because you watched X" suggestions
**Implementation**:
- Enhancement to existing TMDB recommendations
- Add "Because you watched [X]" section using watch history
- Collaborative filtering based on library (simpler than ML)

### 4. External Ratings Display
**Goal**: Show IMDB/Rotten Tomatoes ratings alongside TMDB
**Implementation**:
- TMDB already provides external_ids (IMDB, Freebase) and external_ratings (RT, Metacritic)
- Display in detail pages (MoviePage, TVPage) alongside TMDB rating
- Show red tomato and audience score badges

### 5. Offline Browsing
**Goal**: Browse cached content when offline
**Implementation**:
- Enable offline mode that uses cached TMDB responses
- HomePage shows cached items with "offline available" badge
- Cache search results for offline search

### 6. Queue Management (Replace Saved)
**Goal**: Replace sidebar Saved section with Queue
**Implementation**:
- Modify Sidebar "saved-thumb" to be queue items
- Add "Play Next" button that plays head of queue
- Drag-to-reorder existing in Sidebar.jsx
- Add queue actions: clear, add to queue, remove from queue
- Context menu to change queue order

### 7. Watch Party (Simple)
**Goal**: Share watch position with friends
**Implementation**:
- Generate shareable URL with timestamp
- WebSocket server for real-time sync (optional v2)
- Simple version: Copy current position link

### 8. Trakt.tv Integration
**Goal**: Sync watch history externally
**Implementation**:
- Add Trakt.tv API integration in settings
- Scrobble watching/completed to Trakt
- Import history from Trakt

### 9. Global Keyboard Shortcuts
**Goal**: Control playback when app is minimized
**Implementation**:
- Register global shortcuts with Electron
- Media keys support (play/pause/next/prev)
- Electron globalShortcut API

### 10. Picture-in-Picture (Floating Player)
**Goal**: Floating mini player
**Implementation**:
- Detachable player window
- Picture-in-Picture API or separate BrowserWindow
- Compact view option in player overlay

### 11. Better Continue Watching
**Goal**: Per-season resume position
**Implementation**:
- Store season-level progress alongside episode
- "Continue Season" instead of just "Continue Watching"
- Show which episode to resume in UI

### 12. Search Filtering
**Goal**: Filter by year, genre, rating in DiscoverPage
**Implementation**:
- Add filter dropdowns to DiscoverPage header
- Client-side filtering for loaded results
- Year range, genre multi-select, rating threshold

### 13. Notification System
**Goal**: New episode alerts
**Implementation**:
- Electron notifications for new episodes
- Background check on interval
- System tray icon with badge

---

## Implementation Order

1. **Auto-Play Next** - Modify player/event handlers in TVPage
2. **Queue (Replace Saved)** - Modify Sidebar saved list
3. **Better Continue Watching** - Season-level progress storage
4. **Search Filtering** - DiscoverPage enhancements
5. **External Ratings** - TMDB external_ratings display
6. **Global Shortcuts** - Electron globalShortcut
7. **Offline Browsing** - Cached content mode
8. **Watch Party** - Simple share link
9. **Trakt.tv Integration** - Settings + scrobbling
10. **Skip (Live-Action)** - Chapter/skip integration
11. **PiP Player** - Floating window
12. **Notifications** - System tray badge
13. **Better Recommendations** - History-based sections
